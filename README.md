# シフト管理アプリケーション

## 概要

従業員のシフト管理を効率的に行うためのWebアプリケーションです。オーナーと従業員それぞれの権限に応じた機能を提供し、リアルタイムでのシフト更新と自動給与計算を実現します。

## 主な機能

### オーナー機能
- **全従業員のシフト管理**: 従業員全員のシフトを一覧表示・編集
- **従業員管理**: 従業員の登録・編集・削除
- **シフト作成・編集**: カレンダー形式でのシフト管理
- **リアルタイム更新**: シフト変更の即座反映
- **給与計算**: 月別の給与自動計算と表示

### 従業員機能
- **個人シフト確認**: 自分のシフトのみ表示（他者のシフトは非表示）
- **不足シフト確認**: シフトが未設定の日を表示
- **シフト希望提出**: 希望するシフトの提出機能
- **出退勤ボタン**: 実際の出勤・退勤時間を記録
- **給与確認**: 自分の月給を確認

### 共通機能
- **リアルタイム更新**: シフト変更の即座反映
- **重複チェック**: シフト時間の重複を自動検出
- **労働時間計算**: 日別・月別の労働時間自動計算
